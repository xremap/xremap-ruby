# xremap-ruby

Generate an xremap config with a Ruby DSL.

## Installation

```bash
$ gem install xremap
```

This will install `xremap-ruby` command.

## Usage

```bash
$ xremap-ruby example/config.rb > config.yml
$ sudo xremap config.yml
```
